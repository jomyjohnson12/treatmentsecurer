About The Project

The proposed system Secure Treatment using unique ID is a an application for many hospitals, Medical colleges, Primary health centers to identify patients uniquely. This system provides a unique ID for patients and provide better treatment. Using this software patients treatment details, medicine details, back histories are available to doctors. This software provides patients to get op ticket to consult doctors. Each doctors consulting time and duty time are different and the number of patients for each doctors are also different. When the patient register as a patient , the system will provide a unique ID. This ID will show the details of each patients independently. It will eliminate paper usage. During the period of Covid. The paper transactions to other hospitals , Lab, pharmacy are very dangerous to spread decease. The users of this software are,

•  Admin

•  Patients

•  Doctors

•  Pharmacist

•  Lab technicians

•  Guest Users

This software is a server controlled application. Therefore , it is available anywhere without any time limit. The users get the details of patients with a single mouse click. This software is not only applicable to one hospital one district , but for many.

To develop this software, Python is selected as the front-end application, MySql as the back end application and Django as the frame work for development.
